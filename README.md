# Connecta

Connecta is a mini social media application built using the **MERN Stack**. It enables users to create personalized profiles, share posts, follow other users, and engage with the community through likes and comments. The project focuses on delivering a modern, responsive, and interactive social networking experience.

---

## 🚀 Features

- 🔐 Secure User Authentication
- 👤 Personalized User Profiles
- 📝 Create, Edit, and Delete Posts
- ❤️ Like and Unlike Posts
- 💬 Comment on Posts
- 👥 Follow and Unfollow Users
- 📰 Dynamic Home Feed
- 📱 Responsive User Interface
- ⚡ RESTful API Integration

---

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Tokens)
- bcrypt.js

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/connecta.git
cd connecta
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder and add:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
